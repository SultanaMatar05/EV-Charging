# ChargeMate ⚡🚗
A simple EV (Electric Vehicle) Charging Website built using HTML, CSS, and JavaScript.

## 📌 Overview
ChargeMate is a front-end web project that simulates a basic electric vehicle charging platform.  
It includes user authentication pages like Login and Sign Up with a clean and responsive UI design.

## ✨ Features
- 🔐 Login page
- 📝 Sign up / Create account page
- 📱 Responsive design (mobile & desktop)
- 🎨 Simple and modern UI
- 💻 Built with HTML, CSS, JavaScript

## 🎯 Purpose
This project was created for learning and practicing:
- Front-end web development
- UI/UX design basics
- Multi-page website structure
- Form handling and layout design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (basic)
